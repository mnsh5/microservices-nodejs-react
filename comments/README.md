# Microservices with Node JS and React

Build, deploy, and scale an E-Commerce app using Microservices built with Node, React, Docker and Kubernetes

## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Deno, Express

## Run Locally

Start the server

```bash
  deno task dev 
```
